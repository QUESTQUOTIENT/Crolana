Performance Solutions
Solutions identified to improve performance

Render blocking requests CRITICAL

Requests are blocking the page's initial render, which may delay LCP. Deferring or inlining can move these network
requests out of the critical path.

Est savings of 450 ms FCP: 450ms LCP: 450ms

Network dependency tree CRITICAL

Avoid chaining critical requests by reducing the length of chains, reducing the download size of resources, or
deferring the download of unnecessary resources to improve page load.

Low Code Coverage CAN BE IMPROVED

Code coverage measures how much your JavaScript and CSS are used when loading a webpage. High unused code

removing or deferring unused parts can improve your site's performance and user experience.
